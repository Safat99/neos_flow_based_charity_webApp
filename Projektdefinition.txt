Projekt "Charity"
Das Projekt umfasst eine Webseite, auf der man nach der Eingabe eines Spendencodes eine Organisation auswählen kann, an die gespendet werden soll.

Layout: https://xd.adobe.com/view/8203e2bd-6064-412e-80da-c167004c107b-d750/
Über Klick auf "</>" können Schriftgrößen, Farben etc. ausgelesen und Bilder heruntergeladen werden.

Anforderungen:
- Die Organisationen und deren Eigenschaften (Bild, Beschreibung, Link) sollen in der Settings.yaml konfigurierbar sein.
- Die gültigen Spendencodes sollen durch einen CLI-Command aus einer CSV-Datei (Aufbau siehe codes.csv) ins System importiert werden können.
- Jeder Spendencode soll nur einmal verwendet werden können.
- Eine Auswertung zum Spendenstand soll durch einen CLI-Command generiert werden können. Form und Inhalt der Auswertung sind eigenständig zu definieren.
- Die Links im Footer (Impressum & Datenschutz) sollen auf eigene Seiten zeigen. Diese benötigen keinen Inhalt, das Wort "Impressum" bzw. "Datenschutz" reicht aus, deswegen sind diese nicht im Layout vorhanden.

Folgende Software-Versionen sollen verwendet werden:
- PHP: 8.2
- Neos.Flow: 8.3
- Bootstrap: 5.3

---- translation ------
Project "Charity"
The project includes a website where you can select an organization to donate to after entering a donation code.

Layout: https://xd.adobe.com/view/8203e2bd-6064-412e-80da-c167004c107b-d750/
By clicking on "</>" you can read font sizes, colors, etc. and download images.

Requirements:
- The organizations and their properties (image, description, link) should be configurable in Settings.yaml.
- The valid donation codes should be able to be imported into the system using a CLI command from a CSV file (structure see codes.csv).
- Each donation code should only be used once.
- An evaluation of the donation status should be able to be generated by a CLI command. The form and content of the evaluation must be defined independently.
- The links in the footer (imprint & data protection) should point to your own pages. These do not require any content, the word “imprint” or “data protection” is sufficient, which is why they are not included in the layout.

The following software versions should be used:
- PHP: 8.2
- Neos.Flow: 8.3
- Bootstrap: 5.3
